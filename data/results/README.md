# Results Folder

This folder is reserved for new backtest outputs.

Suggested structure:

- `backtests/`: equity, trades and metrics files.
- `walk_forward/`: walk-forward outputs.
- `robustness/`: bootstrap/monte-carlo robustness runs.
- `risk_scenarios/`: stress and scenario matrices.
- `reports/`: markdown and json summary reports.
- `plots/`: generated figures.

Old outputs were moved to:
